# AI Scheduler

An automated scheduling program through the use of a personlized AI assistant. Automatic schedule integration and rearrangement<br/>

# The Problem

Modern schedulers are innefficient and require constant modifications from the user<br/>
- Reorganizing events
- Defining key events
- Recurring events not based on days of the week

# The Solution

The AI assistant is able to learn from your preferences.<br/>
- Offers suggestions based on your schedule to fit in personal events
- Blacklists time intervals
- Prioritizes ideal times over times close to blacklisted
- Plans around key events
- Automatically schedules new events from chat boxes
- Schedules do not have to be based on days of the week and are completely customizable
